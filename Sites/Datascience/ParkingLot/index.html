Parking - analyse en temps réel

Le développement est basé sur la bibliothèque Open Computer Vision (OpenCV)

On utilise une technique de masque appliqué sur l'image de la caméra pour isoler chaque place de parking

Chaque petit rectangle est présenté à un modèle d'IA de type Classifier binaire réalisé avec un réseau de neurones.

Le modèle est entraîné pour dire si oui ou non la place de parking est libre. Le modèle a été entraîné avec des centaines d'images de places de parking occupées ou vides.
